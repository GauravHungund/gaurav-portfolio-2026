# Portfolio Projects Overview

This document contains a detailed overview of the projects showcased in Gaurav Hungund's portfolio. Each project highlights specific technical skills ranging from full-stack development to AI agent orchestration.

## VoiceCoach (2025)
**Category:** AI / Agentic Systems
**Awards:** Best Use of Neo4j (Autonomous Agents Hackathon)
**Tagline:** An AI that listens, watches, and adapts — so every session makes you sharper.
**Overview:** VoiceCoach is a multi-modal autonomous interview coach. The system audits voice for stress (Modulate), analyzes the camera frame for posture (Reka Vision), extracts skills (Pioneer NER), researches the company live (Yutori), and stores every answer in a Neo4j knowledge graph.
**Tech Stack:** React, TypeScript, Vite, Tailwind CSS, FastAPI, Python, Modulate, Reka Vision, Pioneer NER, Fastino, Neo4j, Yutori.

## Village Simulation / Farmer OP (2025)
**Category:** AI / Simulation
**Tagline:** A living world where agents decide, build, and evolve without human instruction.
**Overview:** Autonomous agent-driven world on a 40x40 dynamic grid. An Oracle agent governs the environment while a Builder agent gathers resources and constructs structures. They run a continuous loop of evaluation and execution with generated tile assets via Freepik API.
**Tech Stack:** React, Vite, Python, Flask, Airia, Fastino, Freepik API.

## Dungeon Forge (2025)
**Category:** AI / Full-Stack
**Awards:** Best Use of Airia Platform (The Future of Agents Hackathon)
**Tagline:** Co-create adventures in real time with an AI Dungeon Master.
**Overview:** A collaborative storytelling platform where players co-create adventures with an AI Dungeon Master. Airia orchestrates the narrative agents, Stack AI handles story logic, OpenAI generates prose, and ElevenLabs voices the scenes.
**Tech Stack:** React, Tailwind CSS, Vite, Flask, FastAPI, Airia, Stack AI, OpenAI, ElevenLabs.

## MicDrop AI (2024)
**Category:** AI / Voice Tech
**Awards:** Best Use of Lightpanda (Production Agents Hackathon)
**Tagline:** Researches, writes, critiques, and speaks a podcast — end to end, no human scripting.
**Overview:** Fully autonomous podcast generator producing two-host conversations. It researches with Lightpanda and Playwright, writes dialogue via Claude, stores memory in Redis, and voices with ElevenLabs — complete with a self-critique loop.
**Tech Stack:** Claude, Lightpanda, Playwright, Redis, ElevenLabs, Flask, Python, React.

## EmoTune (2025)
**Category:** AI / Music Tech
**Tagline:** Translate your emotional state into perfectly matched music.
**Overview:** An emotion-aware recommendation platform pairing emotional state with personalized music and content suggestions to gently shift or validate the user’s mood.
**Tech Stack:** React, Tailwind CSS, Emotion Detection API, Music Recommendation Engine.

## Odyssey (2025)
**Category:** Travel Tech / AWS
**Tagline:** Plan trips intelligently. Preserve memories beautifully.
**Overview:** Smart travel platform combining AI trip planning with memory journaling. It features serverless AWS S3 storage for images (retrieved via signed URLs) and Lambda functions handled by an API Gateway.
**Tech Stack:** React, Vite, Tailwind CSS, React Router, AWS API Gateway, AWS Lambda, AWS S3.

## TaskChain (2024)
**Category:** Mobile / Flutter
**Tagline:** Build better habits through social accountability and streaks.
**Overview:** Cross-platform mobile app tracking habit streaks through social accountability and friend invites. Built with Flutter.
**Tech Stack:** Flutter, Dart, Material Design 3.

## SilentSignal (2024)
**Category:** AI / Safety Tech
**Tagline:** Privacy-first AI that detects emotional abuse patterns and provides crisis support.
**Overview:** Hybrid AI system combining rule-based detection with Nemotron-3 reasoning for emotional abuse pattern recognition, processing everything securely in-memory.
**Tech Stack:** Nemotron-3, RAG Pipeline, Python, FastAPI, Streamlit, WhatsApp API.
